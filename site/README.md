# BPD Healthcare — Astro site

This is the Astro project for the BPD Healthcare static site. See the [root README](../README.md) for full setup and deployment instructions.

## Quick start

```bash
npm install
npm run dev       # dev server at localhost:4321
npm run build     # production build → dist/
npm run preview   # preview production build
```

> **Note:** The `src/data/` JSON files are generated by the parser script at the repo root. Run `node ../scripts/parse-export.mjs` from this directory (or `node scripts/parse-export.mjs` from the repo root) to regenerate them from the WordPress XML export.
